# Banking Management System (Java)

## Project Overview
The Banking Management System is a console-based Java application developed to simulate basic banking operations.  
The project focuses on implementing core business logic using Java and Object-Oriented Programming concepts.

It allows users to create accounts, deposit money, withdraw funds, transfer money between accounts, and check account balances.

---

## Technologies Used
- Java (JDK 8 or above)
- VS Code
- Git & GitHub

---

## Features
- Create new bank accounts
- Deposit money into an account
- Withdraw money with balance validation
- Transfer funds between accounts
- Check account balance
- Menu-driven console interface
